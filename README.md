# Variáveis em Python
Possui basicamente as seguintes caracteristicas :

É a identificação de uma localização na memória RAM. É utilizada para armazenar um valor em memoria temporariamente para ser utilizada durante a execução do nosso código.

```
Identificação (Nome)
Endereço físico
Tipo (Int, Float, String, Bool)
Tamanho
Valor
```

Podes utiliza a função type() para verificares o tipo das variáveis. Ex: type("17")
